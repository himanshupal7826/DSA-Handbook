## Daily routine
1. **15 min** — review yesterday's pattern cheat sheet.
2. **45 min** — study today's pattern page end-to-end.
3. **60 min** — solve the practice set (Easy→Medium→Hard).
4. **Mark complete** on the pattern page; bookmark anything shaky.

## Weekly checkpoint
- Re-solve 3 random past problems *timed*.
- Run the Pattern Selector on 2 unseen problems.
- Update your progress dashboard.
