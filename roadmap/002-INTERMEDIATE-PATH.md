# 📈 Intermediate Path (Patterns 20–52)

**Goal:** log-time search, interval reasoning, and the linear structures (stacks, queues, heaps, linked lists) that power most *Medium* problems.

| Week | Patterns | Focus |
|------|----------|-------|
| 1 | 20–24 Binary Search & bounds | Invariant + boundary bias |
| 2 | 25–28 BS on answer / rotated / peak / monotonic | Predicate search |
| 3 | 29–33 Intervals | Sort + sweep |
| 4 | 34–39 Monotonic Stacks | Next/Previous greater |
| 5 | 40–47 Queues & Heaps | Window max, Top-K, two heaps |
| 6 | 48–52 Linked Lists | Reversal, cycles, dummy node |

## Milestone checklist
- [ ] I can write `lower_bound`/`upper_bound` without off-by-one bugs.
- [ ] I recognize "minimize the maximum" → binary search on answer.
- [ ] I can build a monotonic stack for next-greater in one pass.
- [ ] I can merge K sorted lists with a heap.
