# 🚀 Advanced Path (Patterns 53–88)

**Goal:** trees, graphs, backtracking, dynamic programming, and greedy — the heart of *Medium/Hard* and Senior interviews.

| Week | Patterns | Focus |
|------|----------|-------|
| 1 | 53–60 Trees | DFS/BFS, height, diameter, LCA, tree DP |
| 2 | 61–64 Graph traversal, topo sort, union-find | Connectivity & ordering |
| 3 | 65–68 Dijkstra, Bellman-Ford, Floyd-Warshall, MST | Weighted graphs |
| 4 | 69–73 Backtracking | Subsets, permutations, N-Queens |
| 5 | 74–80 DP I | Knapsack family, LIS, LCS, string DP |
| 6 | 81–88 DP II & Greedy | Grid/tree/state-machine DP, greedy |

## Milestone checklist
- [ ] I choose BFS vs DFS vs Dijkstra by the weight/structure of the graph.
- [ ] I can derive a DP recurrence and convert it to bottom-up + space-optimized.
- [ ] I can prove a greedy choice is safe (exchange argument).
- [ ] I can write the backtracking template (choose → explore → un-choose).
