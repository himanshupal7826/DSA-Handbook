# 🌱 Beginner Path (Patterns 01–19)

**Goal:** think in *O(n)* and master arrays, strings, hashing, two pointers, and sliding windows — the bedrock of 60% of interview questions.

## Week-by-week
| Week | Patterns | Focus |
|------|----------|-------|
| 1 | 01 Frequency Counter, 02 HashMap Lookup | Trade space for time |
| 2 | 03 Prefix Sum, 04 Difference Array, 05 Sorting | Precompute & reorder |
| 3 | 06–07 Two Pointers, 08 Fast/Slow, 09 Dutch Flag | Pointer movement |
| 4 | 10–12 Three/Four Sum, Container | Pairs & triplets |
| 5 | 13–16 Sliding Window (fixed/variable/longest/shortest) | Window invariants |
| 6 | 17–19 Anagram / Distinct / Subarray Sum windows | Window + counting |

## Milestone checklist
- [ ] I can spot when a hash map removes a nested loop.
- [ ] I can convert a range-sum question to prefix sums instantly.
- [ ] I can write the variable sliding-window template from memory.
- [ ] I can solve Two Sum, 3Sum, and Longest Substring without hints.
